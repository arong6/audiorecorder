# 本地音频录制器

一个完全在浏览器中运行的本地音频录制工具，可以录制、播放、保存音频文件到本地。

## 功能特性

- 🎤 **高质量录音**：使用Web Audio API进行高质量音频录制
- 📊 **实时可视化**：录音时显示音频波形可视化
- ⏱️ **计时器**：显示录音时长
- 💾 **本地存储**：录音文件保存在浏览器本地存储中
- 📁 **文件管理**：查看、播放、下载、删除录音文件
- 🎨 **响应式设计**：适配各种屏幕尺寸
- ⌨️ **键盘快捷键**：
  - 空格键：开始/停止录音
  - ESC键：停止录音

## 使用方法

1. 打开 `index.html` 文件（双击或在浏览器中打开）
2. 点击"开始录制"按钮（或按空格键）
3. 授予麦克风访问权限
4. 开始录音，可以看到实时音频可视化
5. 点击"停止录制"按钮（或按空格键/ESC键）结束录音
6. 点击"保存录音"将录音保存为MP3文件
7. 在"录音文件列表"中可以管理所有录音文件

## 技术说明

- 使用 **MediaRecorder API** 进行音频录制
- 使用 **Web Audio API** 进行音频处理和可视化
- 使用 **localStorage** 保存录音文件到本地
- 录音格式：WebM (Opus编码)，下载为 `.webm` 文件
- 完全在浏览器中运行，无需服务器

## 注意事项

1. 录音文件保存在浏览器本地存储中，清除浏览器数据会删除所有录音
2. 需要HTTPS环境或localhost才能访问麦克风
3. 每个浏览器的本地存储限制不同（通常5-10MB）

## 自定义选项

- 录音质量设置
- 文件命名格式
- 可视化效果
- 本地存储策略
